# Finans - Plataforma de Finanças Pessoais

## Descrição
O **Finans** é uma aplicação web responsiva desenvolvida para ajudar usuários a organizarem suas finanças pessoais de forma simples e intuitiva. A plataforma oferece funcionalidades como categorização de despesas, alertas de contas a pagar e relatórios visuais para uma melhor gestão financeira.

## Tecnologias Utilizadas
- **HTML5**: Estrutura do site
- **CSS3**: Estilização personalizada
- **Bootstrap 4.1.3**: Framework CSS para responsividade e componentes prontos
- **Font Awesome**: Ícones para melhor experiência visual
- **JavaScript (jQuery, Popper.js e Bootstrap JS)**: Funcionalidades dinâmicas

## Estrutura do Projeto
O projeto está organizado em diferentes seções:

### 1. **Cabeçalho e Navegação**
- Barra de navegação responsiva com links para diferentes seções (Home, Recursos, Benefícios, Preços e Login)
- Logotipo da aplicação

### 2. **Seção Principal (Home)**
- Chamada para ação com destaque para o objetivo da ferramenta
- Formulário para cadastro de e-mail
- Links para download da versão mobile (Android e iOS)

### 3. **Recursos e Benefícios**
- Explica como a ferramenta ajuda no controle financeiro
- Gráficos simples para visualização de receitas e despesas
- Alertas de contas a pagar e receber

### 4. **Diferenciais**
- Facilidade de uso
- Economia de tempo
- Suporte amigável para dúvidas

### 5. **Rodapé**
- Links rápidos para as principais páginas
- Ícones de redes sociais

## Como Utilizar

   ```bash
   git clone https://github.com/seu-usuario/finans.git
   ```
2. Abra o arquivo `index.html` em seu navegador.
